# Support

## Where to ask

- **Bug 或回归**：在受影响仓库使用 Bug Report 模板创建 Issue。
- **功能建议**：使用 Feature Request 模板，说明实际问题、使用场景与替代方案。
- **使用问题**：先查阅目标仓库 README、文档、示例和已有 Issue；若仍无法解决，再创建带最小复现的问题。
- **安全漏洞**：遵循 [SECURITY.md](SECURITY.md)，不要公开报告。

## What to include

请提供仓库版本或 commit、运行环境、最小配置、复现步骤、预期与实际结果，以及已脱敏的相关日志。没有足够上下文的问题可能会被要求补充信息。

社区支持基于维护者和贡献者的可用时间，不承诺响应时限。请勿在多个仓库重复提交同一问题，也不要公开发布密钥或敏感数据。
